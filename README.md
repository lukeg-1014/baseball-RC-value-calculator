# baseball-RC-value-calculator
Uses baseballpy to scrape baseball player data based on your input. Uses eBay data to gather rookie card data and calculates the average price of a rookie card and whether or not it is worth it to buy based off of stats and card price

To begin run this in your terminal:
pip install -r requirements.txt
playwright install chromium

Then run baseball.py on your computer.
Make sure you have a valid MLB player!
